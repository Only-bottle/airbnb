# airbnb
Clone of the Airbnb Backend built with Django, REST Framework and Strawberry GraphQL

# How to use

## 1. Run Virtual Enviroment
```shell
poetry shell
```

## 2. Migrate DB
```shell
python manage.py migrate
```

## 3. Run Server
```shell
python manage.py runserver
```

## 4. Create Super User
```shell
python manage.py createsuperuse
```
